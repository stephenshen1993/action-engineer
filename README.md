# 行动工程师（Action Engineer）

这是一个关于：

「程序员在 AI 时代重新寻找长期价值」的长期内容项目。

项目核心并不是：
- AI教程
- 流量运营
- 创业鸡汤

而是记录：

一个传统工程师，
如何从“出售时间”
逐渐转向“构建长期资产”的真实过程。

---

# 项目目标

长期记录以下主题：

- AI时代的职业焦虑
- 工程师思维惯性
- 行动力与过度准备
- 内容创作过程
- 长期主义
- 个人成长
- 资产化探索
- AI协作工作流
- 普通人的转型尝试

---
